# FlutterKaigi

本リポジトリは FlutterKaigi に関する有志が集まったリポジトリです。

## FlutterKaigi とは

FlutterKaigi は日本における Flutter の開発者向けカンファレンスです。

以下 2 点を目的にしています。

- エンジニア視点で Flutter 技術情報を共有する
- エンジニア同士コミュニケーションをとる
